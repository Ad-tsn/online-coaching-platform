# Online Coaching Platform

Online coaching booking platform built with Vue.js, Supabase, Cal.com, and Stripe.

This application allows users to select an offer, book a session, complete a checkout process, and leave a verified review linked to a paid order.

## Features

- Multi-step checkout flow (offer selection → booking → user info → payment)
- Real-time booking integration with Cal.com
- Secure payments via Stripe
- Webhook handling (Stripe & Cal.com)
- Dynamic pages generated from database content
- Verified reviews linked to completed paid orders

## Tech Stack

- Vue.js
- Supabase (Database & Auth)
- Stripe (Payments)
- Cal.com (Scheduling)
- JavaScript
